SetCreator

SetCreator(string creator [, boolean isUTF8])

Description

Définit le créateur du document. Il s'agit typiquement du nom de l'application qui génère le PDF.

Paramètres

creator
Le nom du créateur.
isUTF8
Indique si la chaîne est encodée en ISO-8859-1 (false) ou en UTF-8 (true).
Valeur par défaut : false.

SetAuthor, SetKeywords, SetSubject, SetTitle
